# Synk

Synk is a mobile app built with Expo and React Native. It helps users reduce social scrolling with planned blocking moments, preparation screens, habit tracking, iOS widgets, and a Dynamic Island / Live Activity demo.

## Tech Stack

- Expo 54
- React Native 0.81
- React 19
- TypeScript
- i18next for French / English translations
- WidgetKit / ActivityKit scaffolding for iOS widgets and Dynamic Island
- Android native widget scaffolding through an Expo config plugin

## Requirements

- Node.js
- npm
- Expo CLI through `npx`
- Xcode for iOS builds
- Android Studio for Android builds
- CocoaPods for iOS native dependencies

For iOS widgets and Dynamic Island, use a real iPhone with Dynamic Island or an iPhone Pro simulator. Live Activities require iOS 16.1 or newer.

## Installation

From the repository root:

```sh
cd mobile/stop-your-scroll
npm install
```

## Start The App

Start the Expo dev server:

```sh
npm start
```

Then choose a platform from the Expo terminal UI.

## Run On iOS

For a normal iOS native run:

```sh
cd mobile/stop-your-scroll
npm run ios
```

If the iOS native project is missing or stale, regenerate it:

```sh
npx expo prebuild --platform ios
```

Then run again:

```sh
npm run ios
```

You can also open the workspace directly in Xcode:

```sh
open ios/Synk.xcworkspace
```

## Run On Android

## iOS Dynamic Island / Live Activities

The app includes a Live Activity bridge at:

```txt
mobile/stop-your-scroll/src/native/sysLiveActivity.ts
mobile/stop-your-scroll/plugins/withSysIOSLiveActivity.js
```

The Live Activity starts automatically when the preparation screen opens:

```txt
mobile/stop-your-scroll/src/screens/PrepBlockScreen.tsx
```

To test it:

1. Build and run the app on an iPhone with Dynamic Island, or a compatible iPhone Pro simulator.
2. Open the preparation screen.
3. Send the app to the background.
4. Long-press the Dynamic Island to see the expanded view.

Notes:

- The Dynamic Island does not appear while the app is foregrounded.
- Live Activities must be enabled in iOS Settings for the app.
- On devices without Dynamic Island, the activity appears on the Lock Screen instead.
- ActivityKit requires iOS 16.1 or newer.

## iOS Widget Extension

WidgetKit code is scaffolded here:

```txt
mobile/stop-your-scroll/native-widgets/ios/SysWidget.swift
```

After prebuilding iOS, add or verify a Widget Extension target in Xcode. The widget bundle should include:

```swift
SysWidget()
if #available(iOS 16.1, *) {
    SysLiveActivityWidget()
}
```

More details are in:

```txt
mobile/stop-your-scroll/native-widgets/README.md
```

## Android Widget

The Android widget is generated by this Expo config plugin:

```txt
mobile/stop-your-scroll/plugins/withSysAndroidWidget.js
```

Regenerate native Android files with:

```sh
cd mobile/stop-your-scroll
npx expo prebuild --platform android
```

Then build/run Android and look for the `SYS` widget in the launcher widget picker.

## Troubleshooting

If native files are out of sync:

```sh
cd mobile/stop-your-scroll
npx expo prebuild --platform ios
npx expo prebuild --platform android
```

If iOS dependencies are stale:

```sh
cd mobile/stop-your-scroll/ios
pod install
```

If Xcode cannot find generated Live Activity files, run the iOS prebuild again and reopen:

```sh
cd mobile/stop-your-scroll
npx expo prebuild --platform ios
open ios/Synk.xcworkspace
```

If Dynamic Island does not show:

- Use a compatible iPhone or simulator.
- Open the preparation screen first.
- Put the app in the background.
- Check that Live Activities are enabled in iOS Settings.

## Notes

The `ios/` and `android/` folders are generated native projects. Prefer changing Expo config plugins, app code, and files under `native-widgets/` when possible so native changes can be reproduced after a prebuild.
